36.2.9 Non-POSIX Extensions ¶

In addition to implementing the POSIX API for threads, the GNU C Library provides additional functions and interfaces to provide functionality not specified in the standard.

  • Setting Process-wide defaults for thread attributes
  • Controlling the Initial Signal Mask of a New Thread
  • Thread CPU Affinity
  • Wait for a thread to terminate
  • Thread Names
  • Detecting Single-Threaded Execution
  • Restartable Sequences
